template<class Tret, class Tclass, class Targ0, class Targ1, class Targ2, class Targ3>
class goFunctor4< Tret, Tclass, Targ0, Targ1, Targ2, Targ3 >

Member function representation for class members with 4 arguments.

The first template argument is the return type, the second is the class type, the others are member function argument types.
