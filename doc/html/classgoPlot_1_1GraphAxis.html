goPlot::GraphAxis::GraphAxis ( int  index = 0,
double  l = 0.0,
double  u = 1.0 
)

Constructor.

Makes an axis extending from l to . The index indicates which axis it is --- in 2D, 0 indicates the x axis, 1 indicates the y axis. If, for example, you want to make a graph for a sine function in the range from 0 to 2 pi, you can set axis 0 to l = 0 and u = 2 * M_PI, and axis 1 to l = -1 and u = 1.

Parameters:
indexThis axis' index
lLower end of range
uUpper end of range
